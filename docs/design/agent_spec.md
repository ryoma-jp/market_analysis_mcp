# 技術動向調査エージェント（CES等）: Agent Spec

## 目的
CES 2026 などのイベントで「発信された技術動向」を短時間で収集・要約し、
- 重要トピックの全体像
- 競合/プレイヤー別の動き
- 製品・技術の成熟度（PoC/製品化/量産）
- 影響（市場・規制・サプライチェーン・実装面）
を、URL付きで再現可能に提示する。

## 想定ユーザー
- 事業開発/プロダクト: 次の探索領域を絞り込みたい
- R&D: 実装に必要な技術要素とリスクを掴みたい
- 競合調査: 企業/カテゴリー別に動向を追いたい

## 非目的（スコープ外）
- 有料記事/配布資料の全文再掲、スライドの複製
- 未確認情報の断定（噂・リークは「未確認」と明示）

## 主要ユースケース
1. **イベント日次ブリーフ**: 「今日のCESで何が重要だった？」
2. **テーマ別深掘り**: 例: AI PC/オンデバイスAI、車載、XR、ロボ、スマートホーム、半導体
3. **企業別まとめ**: 例: Qualcomm, NVIDIA, Samsung 等の発信整理
4. **比較**: 「昨年(2025)からの変化」

## 入力（ユーザーが指定できる項目）
- イベント名・年（例: CES 2026）
- 対象期間（例: 2026-01-06〜2026-01-10）
- 重点テーマ（複数）
- 重点企業（複数）
- 出力粒度（速報/標準/深掘り）
- 言語（日本語/英語）

## 収集ソース（例）
- 公式: イベント公式サイト（スケジュール/出展者/プレス）
- 企業: 公式ブログ/プレスリリース/ニュースルーム
- 報道: 複数メディア（偏り防止のため最低2系統）
- 動画: キーノート公式配信（可能ならタイムスタンプ）
- ソーシャル: 速報（ただし検証が必要）

**方針**: 1次情報（公式/企業）を優先し、報道は補強に使う。

## 抽出するメタデータ（各“発信”ごと）
- タイトル
- 発信主体（企業/登壇者）
- 日付
- URL
- カテゴリー（後述タクソノミ）
- 主張（何を発表/示唆したか）
- 根拠（要約。数字は出典付き）
- 根拠の現文（結論・見解に紐づく原文の抜粋。短く、出典URLと位置情報を併記）
- 信頼度（高/中/低）

## タクソノミ（最初の案）
- Compute: AI PC, NPU, GPU, Edge AI
- Mobility: SDV, ADAS, インフォテインメント, 車載SoC
- XR: AR/VR/MR, 空間コンピューティング
- Robotics: ヒューマノイド, 物流, 家庭用
- Connectivity: Wi‑Fi/BT, 5G/6G, UWB
- Home: Matter, センサー, 省エネ
- Display: microLED, OLED, 新形状
- Health: ウェアラブル, デジタルヘルス
- Security/Privacy: 規制, 端末内処理, 機密保護

## 出力フォーマット（推奨）
### 1) エグゼクティブサマリ（10行以内）
- 今年の“変化”を中心に
- 結論/見解には、根拠となる情報の現文（短い抜粋）を必ず添える

### 2) トレンドTop N（例: 5〜10件）
各トレンドは以下を含む:
- 何が起きたか（要約）
- 代表例（企業/製品）
- 影響（短期/中期）
- リスク/不確実性
- 根拠の現文（短い抜粋）
- 参照URL（複数）

### 3) 企業別ハイライト（任意）
- 企業ごとの発表を箇条書き

### 4) “次の調査”提案
- 深掘りすべき論点
- 追加で取るべき一次情報

## 品質要件
- **再現性**: 重要な主張には必ずURL
- **偏り対策**: 重要トピックは複数ソースでクロスチェック
- **鮮度**: 日次更新で差分が分かる
- **根拠提示**: 結論/見解ごとに「根拠の現文（短い抜粋）」を示す
- **著作権/契約順守**: 長文引用や転載は避け、必要最小限の短い抜粋に留める

## 根拠の現文（抜粋）ポリシー
- 目的: 「その結論がどの情報に基づくか」を、読み手が即確認できるようにする
- 粒度: 1〜3文程度の短い抜粋を基本（必要以上に長くしない）
- 上限: 1つの抜粋は最大500文字
- 付帯情報: 抜粋には **出典URL** と **位置情報**（見出し名/段落要約/動画タイムスタンプ）を付ける。位置情報が取れない場合は「不明」と明示する
- 使い分け:
	- 一次情報（公式/企業発表）の抜粋を優先
	- 二次情報（報道）の抜粋は補助として扱い、一次情報がある場合はそちらも併記

## 成果物（MVP）
- 「イベント名 + 期間 + テーマ」→ トレンドTop 5 とURL付き出力
- 収集ログ（URL一覧、取得日時、要約）
